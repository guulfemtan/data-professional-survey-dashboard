# Data Professional Survey Dashboard (Power BI)

This project explores insights from a **Data Professional Survey dataset** using a Power BI dashboard.  
The goal of this analysis is to better understand trends in the data industry such as salary levels, programming language preferences, job satisfaction, and the perceived difficulty of entering the field.

The dataset contains responses from **630 data professionals** across different countries and roles.

---

## Dashboard Overview

The Power BI dashboard highlights several key aspects of the survey:

- Total number of survey participants
- Average age of respondents
- Average salary by job title
- Favorite programming languages
- Distribution of participants by country
- Work-life balance satisfaction
- Salary satisfaction
- Difficulty of breaking into the data field

The dashboard helps transform raw survey responses into clear and meaningful insights.

---

## Key Insights

**Python dominates programming language preferences**

Python is the most preferred programming language among survey participants, significantly outperforming other languages such as R, JavaScript, and C/C++.

**Data Scientists report the highest salaries**

The analysis shows that Data Scientists have the highest average salary among the listed job roles, followed by Data Engineers and Data Architects.

**Breaking into the data field can be challenging**

Survey responses indicate that entering the data industry is perceived as moderately difficult by many participants.

**Work-life balance satisfaction is higher than salary satisfaction**

Participants reported higher happiness levels regarding work-life balance compared to salary satisfaction.
